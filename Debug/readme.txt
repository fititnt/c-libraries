----------------------------------- How to use ---------------------------------
1. Install one compiler. GNU GCC Compiler recomended
    - Unix system normaly already have one, so is not need
    - Windows:
        - GNU GCC Compiler via MinGW
        - Any other compiler. MS Already have one
2. Use one Command Line Interface
    - Unix ...
    - Mac ...
    - Windows
        - Depends of what compiler is used. Read specific tutorial
        - Power Shell is recomended
        - cmd.exe can be used too
3. Comand used to compile
    > gcc debug.c -o debug.exe
